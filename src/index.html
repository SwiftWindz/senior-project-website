Project Overview

Project Name: Multi-Cloud Monitoring Dashboard Enhance

Sponser: LenelS2

Coach: Bryan Bashum

Description

The goal of this project is to maintain and implement features for an existing internal React.js web application. This application interfaces with multiple cloud platforms including AWS, Azure and Google Cloud. This application serves as a tool with an intuitive UI that is easy to maintain. This application will be used by digital teams and will help provide insight into cloud metrics and logging information. This information will be conveyed via chart and graph components generated by Grafana.
